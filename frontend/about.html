About StockGuru

Your intelligent stock market analysis companion

Project Overview

StockGuru is a comprehensive stock market analysis platform designed to help investors make informed decisions through real-time data visualization and advanced analytics.

This project was developed as part of our college curriculum to demonstrate proficiency in modern web development technologies and financial data analysis.

Key Features

  • Real-time NSE stock price tracking
  • Interactive charts and data visualization
  • Customizable watchlist management
  • Portfolio tracking with P&L calculations
  • User authentication (Login / Register)
  • Historical data comparison
  • Responsive design for all devices

Technologies Used

  • Frontend: HTML5, CSS3, JavaScript (Vanilla)
  • Styling: Custom CSS with Flexbox & Grid
  • Charts: Chart.js (to be integrated)
  • Backend: Node.js + Express.js
  • Database: MongoDB + Mongoose
  • Auth: JWT (JSON Web Tokens)
  • API: NSE India Stock API

Project Goals

Our primary objectives with StockGuru are to:

  • Provide an intuitive interface for stock market analysis
  • Demonstrate practical application of full-stack web development
  • Integrate real-world financial APIs and data sources
  • Create a scalable and maintainable codebase
  • Deliver a professional-grade user experience
